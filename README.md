pokeca-sim
====

Overview

## Description
ポケモンカードの一人回し用シミュレーターです. 
デッキコードから公式サイトの画像をダウンロードし, シミュレーターで一人回しができます.

## Install

`pip install -r requirements.txt`
で関連ライブラリをインストールできます。

## Usage
`python simulate.py --deck_code=xx8Ycc-z91o6X-J8xxx8 --driver_dir="(path to choromedriver)"`
